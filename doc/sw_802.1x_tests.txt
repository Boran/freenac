
Setup on an access point on port 2/22
--------------------------------------
Lets say there is an access point on port 22, first set it to static and assign a trunk with the 
appropriate vlans:

set port membership 2/22 static
  Port 2/22 vlan assignment set to static.
  Spantree port fast start option set to default for ports 2/22.

set trunk 2/22 on
clear trunk 2/22
  Port(s)  2/22 trunk mode set to auto.
  Port(s)  2/22 trunk type set to dot1q.

sw0503> (enable) set trunk 2/22 11-12,15
Vlan(s) 11-12,15 already allowed on the trunk
Please use the 'clear trunk' command to remove vlans from allowed list.


Setting up 802.1x on port 0/2
-----------------------------

logging  192.168.245.40

interface FastEthernet0/2
  switchport mode access
  dot1x port-control auto
  #dot1x default  
  #dot1x reauthentication  
  #dot1x timeout reauth-period 7200
  #dot1x guest-vlan 524
  #dot1x auth-fail vlan 522

dot1x system-auth-control
radius-server host 192.168.245.40 auth-port 1812 acct-port 1813 key 0 inoap
radius-server retransmit 3

aaa authentication dot1x default group NAC 
aaa authentication dot1x NAC_auth group NAC
aaa authorization network default group NAC

aaa group server radius NAC
  server 192.168.245.41
  
  
#aaa authorization network default group radius
#aaa authentication dot1x default group radius
##aaa authentication dot1x group group radius

Notes
------
dot1x
  auth-fail          Configure Authentication Fail values for this port
  control-direction  set the control-direction value
  default            Configure Dot1x with default values for this port
  guest-vlan         Configure Guest-vlan on this interface
  host-mode          Set the Host mode for 802.1x on this interface
  max-reauth-req     Max No. of Retries to supplicant
  max-req            Max No. of Retries to Radius
  port-control       set the port-control value
  reauthentication   Enable or Disable Reauthentication for this port
  timeout            Various Timeouts

check: dot1x auth-fail vlan
