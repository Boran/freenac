Take Control of your LAN with FreeNAC!

FreeNAC provides easy-to-use Virtual LAN assignment, LAN access control (for all kinds of network devices such as Servers, Workstations, Printers, IP-Phone, Webcams...), live inventory and allows documentation of Patch Cabling.
FreeNAC aims to be the OpenSource product of choice for LAN Access Control.

Now you just need to point your switches to this virtual machine (see the switch configuration examples).

Please check our user's guide, technical guide and installation guide for more detailed information.

GUIs

In this VM are included one web GUI and one Windows GUI which will allow you to interact with the system:
    Web GUI interface
    configured according to our web interface installation guide.

    To configure web authentication using the Windows Domain Controller read the web interface installation guide and set the variable

    $ad_auth=true;

    in /opt/nac/web/config.inc.

    Download the Windows GUI, its configuration file and see how to use it.

802.1X

This VM comes also with support for 802.1X authentication using the Windows Domain Controller. To set it up, please read this HOW-TO.

Help about how to configure your switches to work with 802.1X can be found in http://freenac.net/en/techguide/802.1x.
You are using FreeNAC 3.0, which contains revision 1090 from our subversion repository.

The operating system is Ubuntu 7.10 (Gutsy Gibbon) and includes:
  • FreeNAC 3.0
  • Apache 2.2.4
  • MySQL 5.0.45
  • PHP 5.2.3
  • LibXML 2.6.22
  • Samba 3.0.26
  • FreeRadius 1.1.6
The installation was done according to the installation guide.

To get the latest revision (bug fixes) from the release 3.0, in the directory /opt/nac just type: 
svn update
