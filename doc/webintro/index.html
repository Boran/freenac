Take Control of your LAN with FreeNAC!

FreeNAC provides easy-to-use Virtual LAN assignment, LAN access control (for all kinds of network devices such as Servers, Workstations, Printers, IP-Phone, Webcams...), live inventory and allows documentation of Patch Cabling.
FreeNAC aims to be the OpenSource product of choice for LAN Access Control.

Now you just need to point your switches to this virtual machine (see cisco_switch.txt).

GUIs

In this VM are included one web GUI and one Windows GUI which will allow you to interact with the system:
    Web GUI interface
    configured according to README.web2.

    To configure web authentication using the Windows Domain Controller read this HOW-TO and set the variable

    $ad_auth=true;

    in /opt/nac/web/config.inc.

    Download the Windows GUI, its configuration file and see how to use it.

802.1X

This VM comes also with support for 802.1X authentication using the Windows Domain Controller. To set it up, please read this HOW-TO.

Help about how to configure your switches to work with 802.1X can be found in sw_802.1x_tests.txt.
You are using FreeNAC 2.2 RC1, which contains revision 614 from our subversion repository.

The operating system is Suse Linux 10 and includes:
  • FreeNAC 2.2 RC1
  • Apache 2.2.2
  • MySQL 5.0.22
  • PHP 5.2.0
  • LibXML 2.6.22
  • Samba 3.0.23c
  • FreeRadius 1.1.4
The installation was done according to master_server_install.txt and master_server_config.txt.

To get the latest revision (bug fixes) from the release 2.2 RC1, in the directory /opt/nac just type: 
svn update
